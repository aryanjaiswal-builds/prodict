#PRODICT – Smart Student Productivity Planner
📖 Overview

PRODICT is a web-based productivity application designed to help students plan their daily schedules, improve consistency, and overcome low productivity. It acts as a smart companion that organizes tasks, tracks progress, and ensures deadlines are met efficiently.

🎯 Problem Statement

Students often face challenges like procrastination, poor time management, missed deadlines, and lack of structured planning. These issues lead to stress and reduced academic performance.

💡 Solution

PRODICT provides a simple yet powerful platform to:

Plan daily tasks effectively
Maintain consistency in routine
Track progress in real time
Manage deadlines with ease
✨ Key Features
📅 Task scheduling with subject, time, and deadline
✅ Mark tasks as completed
📊 Real-time progress tracking
⏰ Deadline-focused task management
🧠 Smart task organization using scheduling logic
🛠️ Tech Stack
Technology	Purpose
HTML	Structure of the web app
CSS	UI styling and layout
JavaScript	Task handling & local storage
Python (Flask)	Backend API & logic
C++	Task scheduling optimization
⚙️ How It Works
User adds tasks via the web interface
Tasks are stored using browser localStorage
JavaScript manages task updates and progress
Python backend simulates server-side processing
C++ algorithm optimizes task scheduling based on deadlines & priority
📂 Project Structure
index.html – Frontend UI
style.css – Styling and layout
script.js – Core logic and task management
planner.py – Backend logic (Flask)
scheduler.cpp – Scheduling algorithm
tasks.json – Build configuration
settings.json – Environment settings
📊 Progress Logic

Progress is calculated as:

(completed tasks / total tasks) × 100
